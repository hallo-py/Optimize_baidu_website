# 百度优化插件

这个浏览器插件是为了优化百度搜索结果，针对百度的搜索结果去除广告内容，还可以个性化右菜单栏

# 版本

v0.0.1

# 使用方法

在你的浏览器中安装此插件即可

# 需要获取的浏览器权限

1. storage: 用于存储用户设置，用于个性化。
